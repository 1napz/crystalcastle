## 🔄 PR Governance Flow
1. Open PR  
2. feature / fix / docs  
3. Checklist & PR Template  
4. CI/CD Pipeline  
   - CodeQL & Security Checks  
   - Unit Tests & Coverage  
5. Reviewer Checklist  
6. Governance Review  
7. Merge to Main
